# Platform Overview

The UAV system performs all AI inference fully onboard without cloud dependency, integrating:

- SCRFD face detection
- ArcFace recognition
- Real-time tracking
- Embedded GPU acceleration
- Privacy-aware face blurring
- Live visualization
